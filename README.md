# 🔐 Responsive Login & Registration UI (Frontend Only)

## 📌 Project Overview
This project is a responsive Login and Registration user interface built using HTML, CSS, JavaScript, and Bootstrap 5.  
It demonstrates modern UI design with animations, reusable components, and mobile-first development.

## Features

### ✅ Authentication UI
- Login form
- Registration form
- Switch between Login & Register

### ✅ Form Functionalities
- Show/Hide password toggle 👁️
- Password confirmation check
- Basic frontend validation

### ✅ UI Components
- Responsive Navbar
- Footer section
- Form card layout

### ✅ Responsive Design
- Mobile-first approach 📱
- Works on mobile, tablet, and desktop

### ✅ Animations & Effects
- Fade-in animation
- Button hover effects
- Input focus glow
- Smooth transitions
